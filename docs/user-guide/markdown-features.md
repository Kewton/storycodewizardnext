# Markdown機能

StoryCodeWizardのMarkdown表示機能の詳細と使用方法を説明します。

## Markdown機能の概要

StoryCodeWizardでは、AIからの応答を豊富な表現力でMarkdown形式で表示できます：

- **リアルタイムレンダリング**: ストリーミング中もMarkdown解析
- **シンタックスハイライト**: コードブロックのカラーコーディング
- **表示切り替え**: Markdown/Raw Text切り替え可能
- **ダークテーマ最適化**: 見やすい配色設定
