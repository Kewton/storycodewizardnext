# はじめに

本サービスは **フロントエンドに Next.js、バックエンドに NestJS** を採用し、**モノレポ構成**で開発しています。  
パッケージマネージャーとして `pnpm` を使用し、`turbo` によりビルド・実行・チェックの統一管理を行っています。

## ✅ 品質保証戦略

以下の方針に基づいて、コードの品質・セキュリティを保証します。

| 項目               | 内容                                               | 実行コマンド       |
| ------------------ | -------------------------------------------------- | ------------------ |
| 静的解析           | ESLint + `eslint-plugin-security` によるコード検査 | `pnpm lint`        |
| コード整形         | Prettier によるコードフォーマット統一              | `pnpm format`      |
| 型チェック         | TypeScript による厳密な型検証                      | `pnpm check-types` |
| ユニットテスト     | 自動テストによる基本機能の確認                     | `pnpm test`        |
| セキュリティ監査   | `pnpm audit` による依存ライブラリの脆弱性チェック  | `pnpm audit`       |
| 全チェック一括実行 | 上記全体の統合実行                                 | `pnpm check:all`   |

> 🔁 一括実行時にテスト失敗しても処理は継続されます。
